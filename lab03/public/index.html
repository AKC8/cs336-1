    • a. Static files, as stated in the name, are the files that always get served to the client. They are not dynamically loaded, and are necessary for the client to start. Certain files like images are made static so they load on the client instantly, making the app more responsive.
    • b. The server does continue to serve up the hello-world message as expected. The purpose of express is to handle multiple routes and the REST api requests.
